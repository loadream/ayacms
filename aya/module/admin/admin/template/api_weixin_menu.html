深色按钮代表一级菜单,其上5项为其子菜单;
名称为空的菜单为无效菜单,不在微信中显示;
1、自定义菜单最多包括3个一级菜单，每个一级菜单最多包含5个二级菜单。如果一级菜单没有"动作",须至少包含一个二级菜单

3、创建自定义菜单后，由于微信客户端缓存，需要24小时微信客户端才会展现出来。测试时可以尝试取消关注公众账号后再次关注，则可以看到创建后的效果
4、反复取消关注申请关注后,由于微信缓存原因,需要第二次进入才会显示菜单.
